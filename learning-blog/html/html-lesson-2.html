Lesson 2 - Elements and Tags

Date: 2nd October 2024


Introduction

HTML defines the structure and content of webpages. HTML elements are what creates all of the paragraphs, headings, lists, images and links that make up a typical website.


Elements and Tags

Almost all elements on a HTML page are just pieces of content wrapped in opening and closing tags. Opening tags tell the browser this is the start of a HTML element, and closing tags tell the browser where the element ends. They both consist of angle brackets with a word in them.

For example, the <p> tag would tell the browser that this would be the start of a paragraph element, with </p> marking the end of it.

HTML has a vast list of predefined tags, and it is important to use the correct tags for content for two reasons.

  1. It has a direct impact on how the website is ranked in search engines.
  2. It increases accessibility for users who rely on assitive technologies such as screen readers.
    Using the correct elements for content is called semantic HTML.

Void Elements

Some HTML elements don't have a closing tag, instead just having a single tag such the break element: <br>. These are known as void elements because they are void of any content, due to them having no closing tag. So <br> does not require a </br> closing tag in order for it to function correctly.
